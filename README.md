# Philosophy

Now you might think, why is a single binary both the Agent and the Controller/Manager despite that the Agent can send commands back to the Manager. Well, if the Agent is in Agent mode then it will only do as it is instructed, though, if someone on the other side does get sight of this little bugger on their system and check out how it works and tries to send something back to the C&C then that means C&C needs to also be in Agent mode and also the fact that they might remove this before touching it, unless they're curious. Either way, if you're going to use this for malicious purposes then that's your doing. I only wrote this program for fun and by no means is this program stealthy.
